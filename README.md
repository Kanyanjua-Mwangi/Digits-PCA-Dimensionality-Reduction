# Digits PCA – Dimensionality Reduction

Principal Component Analysis applied to the scikit-learn Digits dataset.

**Key results**:
- 64 dimensions reduced to ~25 components for ≥85% variance
- First 10 PCs capture ~59%, first 25 ~85%
- Demonstrates high redundancy in raw pixel features

See the notebook: `Digits_PCA_Analysis.ipynb`
